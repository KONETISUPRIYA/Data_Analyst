# **Built an Interactive Call Center Performance Dashboard in Excel**
 
Excited to share my latest Excel dashboard project where I analyzed and visualized call center performance metrics using interactive charts and slicers.

# **Key Highlights:**

✔️ KPI Tracking (Total Calls, Resolved, Pending, Escalated)

 ✔️ Resolution Percentage Analysis
 
 ✔️ Agent-wise Call Handling Analysis
 
 ✔️ Daily Call Volume Trend Visualization
 
 ✔️ Interactive Slicers for Dynamic Filtering
 
 ✔️ Call Status Distribution using Donut Chart
 
 ✔️ Clean and User-Friendly Dashboard Design
 
 # **Tools & Skills Used:**

 🔹 Microsoft Excel

 🔹 Pivot Tables
 
 🔹 Pivot Charts
 
 🔹 Slicers
 
 🔹 Data Visualization
 
 🔹 Dashboard Designing
 
 🔹 Data Analysis
 
This project helped me improve my skills in creating business dashboards and transforming raw data into meaningful insights.
